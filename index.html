Visualizing relation between misinformation and public stance on Covid-19 vaccine

Visualizing topic and sentiment

We'll use multiple view to show topics and sentiment side-by-side. The topics will be using point marks. Size of the mark can represent the document count of that topic. Position of the point will represent the relevance to "vaccine".

Sentiment on the other size can be represented using bar charts. Months can be clickable links in the middle to navigate.

Animation

The data can change when clicked on the month name and animation can be used to change the shape of points and bars.

Brushing

Implement brushing to highlight set of topics which will highlight the users who are tweeting the topics.
